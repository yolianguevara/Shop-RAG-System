# Shop-RAG-System

This repository contains a sample shop website integrated with a Retrieval-Augmented Generation (RAG) system.

Structure:
- frontend/  — React + Vite front-end (TypeScript)
- backend/   — Node.js + TypeScript backend (Express) with RAG endpoints
- infra/     — deployment related files (Docker, CI)
- samples/   — sample product data
- docs/      — usage and examples

See docs/USAGE.md for how to run locally.
